Descrição do Projeto

O Etal é um sistema web voltado para o gerenciamento de um evento. O sistema possue funcionalidades de inscrever-se no evento, submeter artigos, aprovar artigo, reprovar artigo. Além disso, o sistema web permite o participante do evento acompanhar o conograma e programação do evento, bem como também conhecer os paletrantes e a cidade que está sediando o evento.
